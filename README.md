# 吾来机器人cmdline客户端 

### mac下安装
```bash
brew install shzy2012/tap/wulai
```


### 使用wulai
```bash
➜  wulai
吾来机器人shell,help帮助,exit退出
>>> help

Commands:
  clear                    clear the screen
  exit                     exit the program
  help                     display help
  init                     初始化吾来客户端
  msgBotResponse           获取机器人回复
  msgBotResponseTask       获取任务机器人回复
  print                    打印 pubkey secret
  userAttributeList        获取用户属性列表
  userAttributeUpdate      更新用户属性
  userCreate               创建用户
```
